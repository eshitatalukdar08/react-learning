React Learning Journey
This repository documents my journey of learning React. Each folder represents a different lesson, showcasing the concepts and code I have worked on.

Lesson Breakdown
Lesson 1: Basic Introduction

An introduction to what React is, its core concepts, and why it's used.

Lesson 2: App & JSX

Learning how to set up the main React application (App.js) and work with JSX, the syntax extension for JavaScript.

Lesson 3: Functional Components

A deep dive into creating and using functional components, a fundamental building block of modern React applications.

Lesson 4: CSS Styling

Exploring different methods to style React components using CSS, including inline styles, CSS modules, and external stylesheets.

Lesson 5: Click Events

Understanding how to handle user interactions by implementing click events on various elements and components.
